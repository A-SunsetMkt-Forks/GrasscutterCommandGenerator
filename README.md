## v0.10.0-M1
- `主页` 页面新增 `语言` 下拉框，可选择简体中文和英语。
- `生成` 页面新增 `摆件` ID分类。


## v0.9.0
命令@UID随版本重构（指令中指定ID改为`@ uid`，因此与旧版本服务端不兼容，不使用`指令含UID`则不影响）

权限管理新增下拉框选择权限（方便选择）

![PermComboBox](Doc/Screenshots/PermComboBox.png)

场景页面新增天气设置（呼风唤雨~）

![SetClimate](Doc/Screenshots/SetClimate.png)
